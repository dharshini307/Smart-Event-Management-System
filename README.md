# Smart Event Management System

A responsive Event Management Website built using HTML, CSS, and JavaScript.

## 🚀 Features

- Responsive Design
- Event Search
- Category Filtering
- Countdown Timer
- Admin Dashboard
- Form Validation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

```
│── style.css
│── main.js
```
